Se quiere implementar un sistema para realizar mediciones de las diferentes propiedades
de los vinos a través de un sistema informático.
Los usuarios deben de ser capaces de registrarse y acceder al sistema a través de un
sistema de autenticación.
Una vez registrados y autenticados los usuarios deben poder recuperar y dar de alta
mediciones del vino con los siguientes datos:
● Año (número)
● Variedad (desplegable)
● Tipo (desplegable)
● color: string,
● temperatura: número
● graduación: número
● ph: número
● observaciones: string

La prueba técnica consiste en el desarrollo del backend y el frontend de la aplicación.
● nodejs: crear el proyecto e implementar servicios de /login, /registry y
/measurements además de modelar la base de datos.
● React: Implementar las ventanas necesarias para la aplicación. A saber:
○ Una primera pantalla de login a la cual se accedera cuando no estás
logueado
○ Una ventana de registro a la que se accederá desde la ventana de login para
dar de alta un usuario
○ Una pantalla de home a la que solo se podrá acceder cuando estás
logueado. En la pantalla de home se mostrará un listado de mediciones
recuperadas a través del servicio de /measurements.

En la prueba se valorará:
● Aplicación de buenas prácticas de código.
● Arquitectura de software.
● Testing
